MolyEcho – Offline-KI-Transkription für Android

Was ist MolyEcho?

MolyEcho ist ein quelloffener Fork von NotelyVoice, entwickelt und optimiert für deutschsprachige Spracherkennung auf Android-Geräten.

Unser Ziel ist eine 100 % private, lokale KI-Transkriptions-Lösung – ohne Clouds, ohne Konten, ohne Kompromisse beim Datenschutz.

Open Source GPL-3.0 100 % Offline Kostenlos

Technologie

MolyEcho basiert auf OpenAI Whisper (eingebettetes Offline-Modell) und ist mit Kotlin Multiplatform und Compose Multiplatform entwickelt.

Projekt auf GitHub

MolyEcho wird von MolyProduction entwickelt und gepflegt. Der gesamte Quellcode ist öffentlich einsehbar.

About

MolyEcho – Offline AI Transcription for Android

What is MolyEcho?

MolyEcho is an open-source fork of NotelyVoice, developed and optimized for German-language speech recognition on Android devices.

Our goal is a 100 % private, local AI transcription solution — no clouds, no accounts, no privacy compromises.

Open Source GPL-3.0 100% Offline Free

Technology

MolyEcho is powered by OpenAI Whisper (embedded offline model) and built with Kotlin Multiplatform and Compose Multiplatform.

Project on GitHub

MolyEcho is developed and maintained by MolyProduction. The full source code is publicly available.
